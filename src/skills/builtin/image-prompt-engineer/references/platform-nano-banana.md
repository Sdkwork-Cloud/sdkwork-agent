# Google Nano Banana Platform Guide

Complete guide for optimizing prompts for Google Nano Banana AI image generation.

## Overview

**Best For:** Quick prototyping, simple concepts, iterative exploration, beginners
**Strengths:** Fast generation, simple interface, accessibility, ease of use
**Approach:** Minimalist - less is more

## Philosophy

Nano Banana works best with **simplicity and clarity**. Unlike other platforms that reward complexity:

- Shorter prompts often work better
- Clear subject focus is key
- Avoid technical jargon
- Simple descriptions get better results

## Prompt Structure

### Minimalist 3-Tier Approach

Unlike the full 6-tier architecture used for other platforms, Nano Banana works best with a simplified structure:

**Tier 1 - Subject (REQUIRED):**
```
Golden retriever puppy
```

**Tier 2 - Key Descriptors (RECOMMENDED):**
```
Golden retriever puppy, playful, sitting in garden, sunny day
```

**Tier 3 - Style/Quality (OPTIONAL):**
```
Golden retriever puppy, playful, sitting in garden, sunny day,
photorealistic, detailed
```

**Maximum Effective Length:** 50-100 words

## Best Practices

### 1. Simplicity First

✅ **Good:**
```
Cat sleeping on windowsill, morning light, cozy
```

❌ **Too Complex:**
```
Domestic feline felis catus in repose upon wooden windowsill,
photons from stellar body Sol entering through glass barrier,
creating chiaroscuro effect with photorealistic rendering...
```

### 2. Clear Subject Focus

✅ **Good:**
```
Red apple on wooden table
```

❌ **Unclear:**
```
Various fruits and vegetables in a scene with natural elements
```

### 3. Specific but Simple

✅ **Good:**
```
Vintage red bicycle leaning against brick wall
```

❌ **Vague:**
```
Something interesting outdoors
```

### 4. Limit Concepts

**Maximum:** 2-3 main ideas per prompt

✅ **Good:**
```
Coffee cup on desk with laptop in background
```

❌ **Too Many:**
```
Coffee cup, laptop, smartphone, notebook, pen, plant, window,
city view, sunrise, rain, person working, cat sleeping...
```

## What Works Best

### Successful Categories

**1. Simple Objects**
- Single item focus
- Clear shapes
- Distinct colors
- Isolated or simple backgrounds

**2. Portraits (Simple)**
- Clear facial features
- Simple expressions
- Uncluttered backgrounds
- Good lighting

**3. Landscapes (Iconic)**
- Recognizable scenes
- Clear horizon
- Simple compositions
- Good color contrast

**4. Food Photography**
- Appetizing presentation
- Simple plating
- Good lighting
- Clean backgrounds

**5. Animals**
- Clear species identification
- Cute/appealing poses
- Simple environments
- Good detail

### Less Successful

**Avoid:**
- Complex multi-subject scenes
- Technical specifications (lens, aperture)
- Very long descriptions
- Abstract concepts
- Multiple conflicting styles
- Extremely detailed architectural scenes
- Complex action sequences

## Migration to Advanced Platforms

### Upgrading from Nano Banana

When your concept is validated in Nano Banana, upgrade to other platforms for refinement:

**Nano Banana → Midjourney:**
```
Original: "Futuristic car, sleek, red, driving"

Upgraded: "Sleek futuristic sports car in metallic crimson red,
aerodynamic curves reflecting sunset light, driving on coastal highway,
photorealistic automotive photography, 8K detailed, --ar 16:9 --v 6.1"
```

**Nano Banana → Jimeng:**
```
Original: "Chinese dragon, golden, flying"

Upgraded: "Majestic golden Chinese dragon (龙) soaring through clouds,
traditional Chinese painting style (水墨画), iridescent scales catching sunlight,
auspicious symbol of prosperity, 国画风格, detailed scales and whiskers"
```

**Nano Banana → Kling:**

```
 lush temperate rainforest surrounding, morning mist rising from water,

```

## Tips for Best Results

### 1. Start Simple
Begin with basic subject + one descriptor, then add if needed.

### 2. Iterate Quickly
Generate multiple simple variations rather than one complex prompt.

### 3. Focus on One Thing
Choose the most important aspect and emphasize that.

### 4. Use Common Terms
Stick to everyday language rather than technical or esoteric terms.

### 5. Clear Intent
Make sure your description clearly conveys what you want.

### 6. Test Variations
Try slight wording changes to see different results.

## Troubleshooting

### Issue: Results Don't Match Description
**Solution:** Simplify prompt, focus on key element

### Issue: Too Abstract
**Solution:** Add "photorealistic" or "detailed"

### Issue: Quality Too Low
**Solution:** Add "high quality" or "detailed"

### Issue: Wrong Style
**Solution:** Explicitly state desired style

### Issue: Confusing Elements
**Solution:** Remove secondary elements, focus on one subject

## Integration Workflow

**Step 2: Refinement (Midjourney/Jimeng)**
