## Traditional Chinese Art Styles

### Ink Wash Painting (水墨画)

**Characteristics:**
- Black ink gradients (shades of gray)
- Minimal color
- Flowing brushstrokes
- Empty space (留白) as important as filled
- Capturing spirit over detail

**Keywords:**
```
水墨画 shuimo hua
水墨风格 ink wash style
淡墨 light ink
浓墨 dark ink
泼墨 splashed ink
写意 xieyi (freehand)
工笔 gongbi (meticulous)
```

**Prompt Example:**
```
Traditional Chinese ink wash painting, 水墨画风格,
mountains emerging from mist, 远山如黛,
minimalist composition with maximum impact,
flowing brushstrokes, 笔触流畅,
留白 empty space creating balance,
古典 Chinese classical aesthetics
```

### Gongbi Style (工笔画)

**Characteristics:**
- Highly detailed
- Fine brushwork
- Vibrant colors
- Precise lines
- Realistic within stylization

**Keywords:**
```
工笔画 gongbi hua
精细 detailed
细腻 delicate
工笔重彩 meticulous brushwork with heavy colors
写实 realistic painting
```

**Prompt Example:**
```
Chinese gongbi painting style, 工笔画风格,
exquisite peony flowers with detailed petals,
精致的细节 fine intricate details,
vibrant traditional Chinese colors,
rich ruby reds and emerald greens,
传统花鸟 traditional flower-and-bird painting
```

### Traditional Landscape (山水画)

**Characteristics:**
- Mountains and water
- Atmospheric perspective
- Scholarly elements (pavilions, bridges)
- Natural harmony
- Philosophical depth

**Keywords:**
```
山水画 shanshui hua (mountain-water painting)
山水 landscape
意境 yijing (artistic conception)
气韵生动 qi yun sheng dong (spirit resonance)
三远 three distances (perspective)
```

**Prompt Example:**
```
Classical Chinese landscape painting, 古典山水画,
towering mountains disappearing into clouds,
waterfall cascading into misty valley,
small pavilion on cliff edge with scholar figure,
三远 perspective creating depth,
气韵生动 with spiritual vitality
```

## Aesthetic Concepts

### Key Chinese Artistic Principles

**意境 (Yi Jing) - Artistic Conception:**
```
意境深远 profound artistic conception,
诗情画意 poetic imagery,
情景交融 scene and emotion blending,
余韵 lingering charm
```

**留白 (Liu Bai) - Empty Space:**
```
大量留白 generous empty space,
虚实相生 void and substance complementing,
空灵 ethereal spaciousness,
简洁 restrained simplicity
```

**气韵 (Qi Yun) - Spirit Resonance:**
```
气韵生动 vibrant spirit resonance,
生动 lifelike vitality,
神韵 spiritual charm,
栩栩如生 vivid and lifelike
```

**含蓄 (Han Xu) - Restrained Implication:**
```
含蓄 restrained and subtle,
委婉含蓄 indirect and implicit,
耐人寻味 thought-provoking,
言有尽而意无穷 meaning beyond words
```
