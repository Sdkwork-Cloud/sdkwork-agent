# 电影视觉语言系统 (Cinematic Visual Language)

## 核心原则

所有提示词必须使用**自然语言描述**，让导演、摄影师、美术指导能一目了然地理解画面。

**禁止行为：**
❌ 技术参数堆砌（"35mm lens f/1.8 shallow depth of field"）
❌ 关键词罗列（"8K, photorealistic, highly detailed"）
❌ 抽象形容词（"beautiful, stunning, amazing"）

**正确做法：**
✅ 画面叙述（"远景中，一位身披斗篷的旅人站在悬崖边"）
✅ 光影描述（"夕阳从画面右侧斜射，在地面拉出长长的影子"）
✅ 氛围营造（"晨雾缭绕，远处的山峰若隐若现"）

## 镜头语言框架

### 1. 景别描述 (Shot Size)

不要用技术术语，用空间感描述：

**远景/全景：**
- "广阔的山谷中，一座孤城矗立在远方"
- "从高空俯瞰，整个城市尽收眼底"
- "海平线上，一艘帆船缓缓驶来"

**中景：**
- "人物腰部以上入画，背景是喧嚣的街市"
- "两人对坐交谈，桌案上的茶具清晰可见"
- "马背上的骑士，马鞍和缰绳细节分明"

**近景/特写：**
- "女子的面容占据画面中心，睫毛上挂着泪珠"
- "老人的手轻抚古琴，指节上的皱纹清晰"
- "剑锋上滑落的一滴血，在画面中格外醒目"

### 2. 机位与角度 (Camera Position)

描述观看位置，而非技术参数：

**视角高度：**
- "从地面仰视，人物显得高大威严"
- "俯视角度，如同飞鸟俯瞰大地"
- "平视，如同站在人物对面"

**观看距离：**
- "仿佛站在三步之外静静观察"
- "隔着人群远远望去"
- "近在咫尺，几乎可以触摸"

**特殊视角：**
- "透过窗棂的缝隙窥视"
- "从水面倒影中看见"
- "仿佛藏在角落里的视角"

### 3. 光影描述 (Lighting)

用自然现象和感受描述光线：

**光源方向：**
- "阳光从画面左侧的窗户倾泻而入"
- "月光从头顶洒落，在地面投下斑驳树影"
- "火光在前方跳动，在脸上映出暖色"

**光线质感：**
- "清晨柔和的光线，如同薄纱笼罩"
- "正午刺眼的阳光，明暗对比强烈"
- "黄昏时分的暖光，给一切镀上金色"

**特殊光效：**
- "阳光穿透云层，形成一道道光柱"
- "烛光摇曳，在墙上投下晃动的影子"
- "霓虹灯在雨后的地面上反射出彩色光斑"

### 4. 运动描述 (Motion)

描述视觉流动，而非技术动作：

**主体运动：**
- "裙摆随风轻扬，如同水波荡漾"
- "马蹄踏起尘土，在身后形成烟幕"
- "落叶缓缓飘落，打着旋儿落在水面"

**环境运动：**
- "炊烟袅袅升起，在屋顶上方飘散"
- "水面泛起涟漪，一圈圈向外扩散"
- "云层在天空中缓缓流动"

**镜头运动（视频）：**
- "视角缓缓推进，从全景逐渐聚焦到人物面容"
- "镜头环绕主体，展现周身环境"
- "视线跟随飞鸟，从地面升至高空"

## 氛围与情绪描述

### 色彩氛围

用感官描述色彩：

**暖色调：**
- "炉火般的暖光，让人感到温暖安心"
- "夕阳将整个画面染成蜜糖色"
- "烛光摇曳，在暗红色的墙壁上投下暖影"

**冷色调：**
- "月光下的青灰色世界，透着寒意"
- "黎明前的蓝调时刻，万物静谧"
- "雪地反射的天光，让画面呈现冷白色"

**对比色调：**
- "橙红的夕阳与青紫的阴影形成鲜明对比"
- "昏暗的环境，唯有手中灯火温暖"

### 情绪关键词

用具体意象传达情绪：

**孤独：**
- "空旷的原野上，一盏孤灯"
- "迷雾中独自行走的背影"

**紧张：**
- "紧绷的肌肉，蓄势待发"
- "阴影中若隐若现的威胁"

**宁静：**
- "水面如镜，倒映着完整的天空"
- "雪落无声，世界一片纯白"

**欢愉：**
- "阳光透过树叶洒下斑驳光影"
- "孩子们奔跑时扬起的笑声"

## 物理真实性规范

### 光影一致性

- 所有阴影方向必须与光源一致
- 反射必须出现在可反射的表面上
- 透明/半透明物体的光线行为要准确

**错误：** "阳光从左边照来，但人物影子向右"
**正确：** "夕阳从右侧低角度照射，在人物左侧拉出长长的影子"

### 空间透视

- 远处物体要模糊、淡化
- 近大远小的比例关系
- 大气透视（远处偏蓝偏淡）

**错误：** "远山和近景一样清晰鲜艳"
**正确：** "远山在晨雾中若隐若现，呈现出淡青色"

### 材质表现

描述材质时考虑光线：

**金属：** "刀身在火光中闪烁，映出周围的环境"
**水面：** "平静的湖面如镜，倒映着完整的天空和云朵"
**织物：** "丝绸长袍在微风中轻轻飘动，反射着柔和的光泽"
**皮肤：** "阳光透过面纱，在脸上投下柔和的光影"

## 风格化描述

### 写实风格

强调真实世界的质感：
"雨后的街道，地面上的积水倒映着霓虹灯，空气中还弥漫着潮湿的气息"

### 动漫风格

强调线条和色彩：
"线条流畅的角色，背景是渐变的柔和色彩，整体呈现出温馨的动漫感"

### 水墨风格

强调笔墨韵味：
"墨色晕染的山水，留白恰到好处，笔触流畅有力，充满东方意境"

### 赛博朋克

强调光影对比和科技元素：
"霓虹灯在雨夜中闪烁，湿润的地面反射着五彩斑斓的光，高楼林立如钢铁森林"

## 提示词结构模板

### 标准格式

```
[景别]，[主体描述]，[环境背景]。
[光影描述]，[色彩氛围]。
[细节焦点]，[材质质感]。
[情绪氛围]。
```

### 示例

**场景：** 古风女子在雨中庭院

**标准提示词：**
```
中景，一位身着青色汉服的古风女子，手持油纸伞，静静伫立在庭院中。
细雨飘洒，从画面左侧的屋檐滴落，在青石板上溅起水花。
女子面容清秀，发髻上几缕青丝被微风吹动。
背景是朦胧的远山和传统建筑，整个画面笼罩在烟雨朦胧的氛围中。
```

**对比 - 旧版（不推荐）：**
```
Ancient Chinese woman, hanfu dress, holding umbrella, standing in courtyard, rain, 85mm lens f/2.8, soft lighting, 8K, highly detailed
```

## 使用指南

1. **先描述看到的画面** - 像在给朋友讲一个场景
2. **从大到小** - 先整体环境，再主体，最后细节
3. **调动感官** - 不只是视觉，还有氛围、情绪
4. **避免清单式罗列** - 用连贯的句子，而非关键词
5. **保持画面感** - 每句话都应该能在脑海中形成画面

**记住：你是在用文字画画，不是在写配置清单。**
