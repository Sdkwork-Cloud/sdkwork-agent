# Input Guidelines

Guide for writing good user inputs.

## Writing Good Input

### DO
- Describe what you see: "A woman standing by the window"
- Include mood: "peaceful morning", "dramatic sunset"
- Mention environment: "in a forest", "on a busy street"
- Add action for video: "walking slowly", "wind blowing"

### DON'T
- List technical specs: "35mm lens, f/1.8"
- Use abstract adjectives: "beautiful", "stunning"
- Over-describe: Keep it to 2-3 sentences
- Ask questions: "Can you make..."

## Input Examples

### Good
```
A samurai standing in bamboo forest at dawn, mist rising from ground
```

### Bad
```
Make me a beautiful picture of a samurai with amazing details and stunning quality
```

### Good
```
Ancient Chinese garden in spring, cherry blossoms falling, peaceful atmosphere
```

### Bad
```
Asian style garden with flowers, 8K, photorealistic, masterpiece, trending on artstation
```
