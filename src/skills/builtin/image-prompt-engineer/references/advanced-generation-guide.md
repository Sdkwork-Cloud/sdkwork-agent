# Advanced Generation Guide
Specialized image generation types, photographic techniques, artistic styles, and advanced concepts.

---

## Specialized Image Types

### 1. Avatars & Profile Pictures (头像/PFP)
**Keywords:** 头像, PFP, profile picture, avatar, Discord, Twitter, social media

**Key Requirements:**
- Close-up or headshot only
- Clear facial features
- Simple background
- Recognizable character
- Square aspect ratio (1:1) recommended

**Prompt Keywords:**
```
Profile picture avatar, close-up portrait, headshot only,
simple clean background, recognizable character, square composition
```

---

### 2. Icons & UI Elements (图标/UI)
**Keywords:** 图标, icon, UI, interface, button, logo, symbol, app icon

**Key Requirements:**
- Simple, clean design
- High contrast
- Minimal detail
- Vector-like style
- Square or 16:9 aspect ratio

**Prompt Keywords:**
```
Flat design icon, minimalist, simple clean lines,
high contrast, vector style, UI element, solid colors
```

---

### 3. Wallpapers (壁纸)
**Keywords:** 壁纸, wallpaper, background, desktop, phone, lockscreen

**Key Requirements:**
- High resolution (4K/8K)
- Wide aspect ratio (16:9 or 21:9 for desktop, 9:16 for phone)
- No central focal point (or subject off-center)
- Clean composition, not too busy
- Room for icons/text

**Prompt Keywords:**
```
4K wallpaper, [scene description], wide composition,
clean aesthetic, no central focal point, room for desktop icons
```

---

### 4. Posters & Covers (海报/封面)
**Keywords:** 海报, poster, cover, book cover, album cover, movie poster

**Key Requirements:**
- Strong focal point
- Clear composition hierarchy
- Space for text (if needed)
- Dramatic visual impact
- Aspect ratio depends on medium

**Prompt Keywords:**
```
Movie poster style, dramatic composition, strong focal point,
professional design, cinematic, space for title text
```

---

### 5. Emojis & Stickers (表情包/贴纸)
**Keywords:** 表情包, emoji, sticker, emoticon, meme, reaction

**Key Requirements:**
- Simple, bold design
- Clear facial expression
- Exaggerated features
- White or transparent background
- Square aspect ratio
- Cartoon style

**Prompt Keywords:**
```
Cartoon emoji sticker, simple bold design, exaggerated expression,
clean white background, square, cute and expressive
```

## Photographic Techniques

### 1. Macro Photography (微距摄影)
**Keywords:** 微距, macro, close-up, extreme close-up, detail

**Key Requirements:**
- Extreme close-up
- Shallow depth of field
- Focus on texture/details
- Small subject (insects, flowers, textures)

**Prompt Keywords:**
```

sharp focus on [subject], detailed texture, professional macro
```
