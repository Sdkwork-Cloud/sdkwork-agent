## Cinematic-Specific Elements

### Dramatic Lighting (Tier 3 Enhanced)

**Key Light Types:**
```
Key light: Primary illumination defining subject
Fill light: Softens shadows
Rim light: Separates subject from background
Background light: Adds depth
Practical lights: Visible sources in scene (lamps, neon, etc.)
```

**Dramatic Patterns:**
```
Chiaroscuro: Strong contrast between light and dark
Low-key lighting: Predominantly dark with selective illumination
High-key lighting: Bright, minimal shadows (comedy, romance)
Motivated lighting: Light appears to come from visible sources
```

**Cinematic Keywords:**
```
dramatic lighting, chiaroscuro, volumetric lighting,
cinematic contrast, motivated lighting, practical lighting,
rim lighting, silhouette, backlit, side-lit, underlit
```

### Cinematic Camera Work (Tier 4 Enhanced)

**Aspect Ratios:**
```
21:9 (2.39:1) - Ultra widescreen, epic scope
16:9 (1.78:1) - Standard modern cinematic
2.35:1 - Anamorphic widescreen
4:3 (1.33:1) - Classic/retro feel
```

**Cinematic Lenses:**
```
Anamorphic: Oval bokeh, horizontal lens flare
Tilt-shift: Miniature effect, selective focus
Vintage: Character, imperfections, warmth
Cooke lenses: Creamy bokeh, cinematic look
```

**Camera Movements:**
```
Dolly: Smooth forward/backward
Tracking: Following subject
Crane: Sweeping vertical movement
Steadicam: Floating, human perspective
Handheld: Documentary, gritty feel
Static: Formal, contemplative
```

**Composition Techniques:**
```
Rule of thirds: Classic cinematic composition
Leading lines: Directing eye through frame
Depth layers: Foreground, midground, background
Frame within frame: Doorways, windows, arches
Dutch angle: Tilted for tension/disorientations
Symmetry: Formal, balanced, often center-framed
```

## Genre-Specific Guidelines

### Film Noir

**Lighting:**
- High contrast (chiaroscuro)
- Low-key lighting
- Strong shadows
- Venetian blind patterns
- Single light sources

**Setting:**
- Urban environments
- Night scenes
- Rain-slicked streets
- Shadows and fog
- Architecture with strong lines

**Keywords:**
```
noir, chiaroscuro, venetian blind shadows, high contrast,
low-key lighting, urban night, rain-slicked streets,
moody atmosphere, cynical mood
```

### Science Fiction

**Lighting:**
- Neon and practical lights
- Harsh artificial sources
- Volumetric beams
- Holographic glow
- Clean, hard shadows or diffuse ambient

**Setting:**
- Futuristic architecture
- High-tech environments
- Space or alien landscapes
- Clean lines or dystopian decay

**Keywords:**
```
sci-fi, futuristic, neon lighting, volumetric beams,
holographic displays, clean lines, dystopian,
cyberpunk, high-tech environment
```

### Western

**Lighting:**
- Golden hour sun
- Natural, motivated sources
- Campfire warmth
- Long shadows
- Dust particles in light

**Setting:**
- Vast landscapes
- Desert environments
- Small towns
- Natural rock formations

**Keywords:**
```
western, golden hour, vast landscape, natural lighting,
desert, long shadows, epic scope, frontier,
spaghetti western style
```

### Horror

**Lighting:**
- Low-key, predominantly dark
- Single practical sources (flashlight, candle)
- Underlighting for unease
- Shadows obscuring details
- Sudden bright moments

**Setting:**
- Confined spaces
- Old buildings
- Woods at night
- Abandoned locations

**Keywords:**
```
horror, low-key lighting, underlit, practical lighting,
shadows, ominous atmosphere, tension, dread,
chiaroscuro, isolated
```

## Common Cinematic Mistakes to Avoid

❌ Over-lit scenes (no shadows)
❌ Unmotivated light sources
❌ Inconsistent lighting direction
❌ Flat, even lighting without contrast
❌ Wrong color temperature for scene
❌ Camera movements without motivation
❌ Aspect ratio that doesn't match content
