# Style Reference
Video style detection and guides.

---

## Style Detection

Detect style from user input keywords:

| Style | Keywords |
|-------|----------|
| Cinematic (Default) | 电影, cinematic, 电影感, movie, film |
| Photorealistic | 写实, 照片, photorealistic, photo, realistic, 真实 |
| Anime | 动漫, anime, manga, 卡通, cartoon |
| Fantasy | 奇幻, fantasy, 魔幻, 魔法 |
| Sci-Fi | 科幻, sci-fi, cyberpunk, 赛博朋克, 未来 |
| Documentary | 纪录片, documentary, real |

---

## Default Style

If NO style specified, use **Cinematic** as default:
- Natural cinematic lighting
- Professional cinematography
- Smooth camera movement
- Professional composition
