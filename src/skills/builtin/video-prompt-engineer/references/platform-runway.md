# Runway 视频生成平台指南

## 平台概述

Runway 是领先的 AI 视频生成平台，提供:
- **Gen-3 Alpha** - 最新的视频生成模型
- **Gen-2** - 成熟的视频生成模型
- **Video to Video** - 视频风格转换
- **Image to Video** - 图片生成视频
- **Motion Brush** - 局部运动控制

## 关键特性

| 特性 | 描述 |
|------|------|
| 最大时长 | 20秒（可扩展） |
| 分辨率 | 1792x1024 |
| 宽高比 | 16:9, 9:16, 1:1, 4:5 |
| 图生视频 | ✅ 支持 |
| 视频到视频 | ✅ 支持 |
| 运动画笔 | ✅ 支持 |
| 首尾帧控制 | ✅ 支持 |

## 提示词指南

### 结构建议

```
[主体/角色], [动作/行为], [环境/背景], [时间/天气], [风格/氛围], [技术参数], [负面提示词]
```

### 示例

**基础示例:**
```
一只金毛犬在沙滩上奔跑，海浪拍打着岸边，阳光明媚，写实风格，4K，高质量
```

**高级示例:**
```
宇航员在月球表面行走，地球在背景中升起，星空璀璨，科幻风格，电影质感，慢镜头，IMAX画质，高质量
```

## 关键参数

| 参数 | 类型 | 范围/选项 | 说明 |
|------|------|-----------|------|
| `duration` | number | 5-20 | 视频时长（秒） |
| `aspect_ratio` | string | "16:9", "9:16", "1:1", "4:5" | 视频宽高比 |
| `model` | string | "gen3", "gen2" | 使用的模型 |
| `motion` | number | 0-10 | 运动强度 |
| `seed` | number | 任意整数 | 随机种子 |
| `watermark` | boolean | true/false | 是否加水印 |

## 最佳实践

### ✅ 推荐做法

1. **清晰的主体描述**
   - 明确说明主要对象
   - 描述主体的特征和状态
   - 说明主体的动作和行为

2. **环境和氛围**
   - 详细描述环境
   - 说明时间和天气
   - 描述光线和色彩

3. **技术质量词**
   - 使用：4K, 8K, high quality, detailed
   - 避免：low quality, blurry, distorted

### ❌ 避免的做法

1. **过多主体**
   - 避免同时出现太多主体
   - 避免过于复杂的场景

2. **快速运动**
   - 避免过于快速的运动
   - 避免复杂的动作序列

3. **负面提示词滥用**
   - 避免负面提示词过多
   - 避免负面提示词与正面描述冲突

## 完整示例

### 文生视频示例

**输入提示词:**
```
一位小提琴手在音乐会上演奏，舞台灯光聚焦，观众在背景中，优雅的氛围，古典音乐，电影质感，4K，高质量
```

**输出预期:**
- 时长：16秒
- 分辨率：1792x1024
- 宽高比：16:9
- 模型：Gen-3 Alpha
- 运动强度：5

### 图生视频示例

**输入:**
- 参考图片：森林风景照
- 提示词：`风吹动树叶，阳光穿过树冠，宁静自然，4K，高质量`
- 运动强度：7

**输出预期:**
- 时长：12秒
- 树叶随风摆动
- 阳光穿透效果
- 自然流畅的运动

### Motion Brush 示例

**输入:**
- 原图：静止的湖面照片
- 提示词：`湖水微波荡漾，4K，高质量`
- 运动区域：湖面区域

**输出预期:**
- 时长：8秒
- 只有湖面有运动
- 其他区域保持静止
- 自然的水波效果

## 常见问题

### Q: 如何让视频更稳定？
A: 降低运动强度，添加 "stable camera" 到提示词。

### Q: 可以延长视频时长吗？
A: 可以使用视频扩展功能，或者将多个视频片段连接起来。

### Q: 如何控制特定区域运动？
A: 使用 Motion Brush 工具，只在需要运动的区域绘制。
