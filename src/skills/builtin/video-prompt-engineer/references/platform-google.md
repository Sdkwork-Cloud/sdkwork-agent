# Google 视频生成平台指南

## 平台概述

Google 提供多个视频生成相关的 AI 工具，包括:
- **Gemini Video Generation** - Gemini 模型的视频生成能力
- **Google AI Studio** - 包含视频生成的开发者平台
- **Google Cloud Vertex AI** - 企业级视频生成服务

## 关键特性

| 特性 | 描述 |
|------|------|
| 最大时长 | 60秒 |
| 分辨率 | 1920x1080 (1080p) |
| 宽高比 | 16:9, 9:16, 1:1 |
| 图生视频 | ✅ 支持 |
| 首尾帧控制 | ✅ 支持 |
| 视频扩展 | ✅ 支持 |

## 提示词指南

### 结构建议

```
[主题/场景描述], [主体/角色], [动作/行为], [环境/背景], [时间/天气], [风格/氛围], [技术参数]
```

### 示例

**基础示例:**
```
一位年轻的创业者在现代办公室内使用笔记本电脑工作，阳光透过大窗户洒进来，温馨的工作氛围，自然光线，4K，高质量
```

**高级示例:**
```
清晨的森林，阳光穿过树叶，一只鹿在小溪边饮水，晨雾弥漫，宁静自然，写实风格，电影质感，慢镜头，4K，60fps
```

## 关键参数

| 参数 | 类型 | 范围/选项 | 说明 |
|------|------|-----------|------|
| `duration` | number | 5-60 | 视频时长（秒） |
| `aspect_ratio` | string | "16:9", "9:16", "1:1" | 视频宽高比 |
| `resolution` | string | "720p", "1080p" | 视频分辨率 |
| `style` | string | "realistic", "cinematic", "animated" | 视频风格 |
| `camera_movement` | string | "static", "pan", "zoom", "dolly" | 相机移动 |

## 最佳实践

### ✅ 推荐做法

1. **详细的场景描述**
   - 包含时间、天气、光线、环境
   - 描述主体的动作和表情
   - 说明整体氛围和情绪

2. **注重细节**
   - 服装细节
   - 背景物体
   - 色彩搭配
   - 光线方向

3. **使用风格词**
   - 写实风格：realistic, photorealistic
   - 电影风格：cinematic, film grain
   - 动画风格：animated, cartoon

### ❌ 避免的做法

1. **过于简略**
   - 避免只有2-3个词的提示词
   - 避免缺乏场景描述

2. **逻辑矛盾**
   - 避免在同一时间出现不同天气
   - 避免不可能的场景组合

3. **敏感内容**
   - 避免违法违规内容
   - 避免侵犯版权的内容

## 完整示例

### 文生视频示例

**输入提示词:**
```
夜晚的城市街道，霓虹灯光闪烁，行人匆匆走过，雨水打湿的路面反射着灯光，赛博朋克风格，4K，高质量
```

**输出预期:**
- 时长：30秒
- 分辨率：1080p
- 宽高比：16:9
- 风格：赛博朋克
- 相机移动：缓慢平移

### 图生视频示例

**输入:**
- 参考图片：城市夜景照片
- 提示词：`霓虹灯光闪烁，车辆行驶，赛博朋克风格，4K，高质量`

**输出预期:**
- 时长：15秒
- 基于图片内容动起来
- 霓虹灯光闪烁效果
- 车辆行驶动画

## 常见问题

### Q: 如何提高视频质量？
A: 使用更详细的描述，包含光线、材质、质感等细节。

### Q: 支持哪些视频风格？
A: 支持写实、电影、动画、赛博朋克、水彩、油画等多种风格。

### Q: 可以控制视频时长吗？
A: 可以，支持5-60秒的视频生成。
