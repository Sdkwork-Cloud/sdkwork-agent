# Script Intelligence Guide

Guide for script processing, evaluation, and optimization.

---

## Input Type Detection

| Type | Detection |
|------|-----------|
| Short Idea | < 50 words, simple description |
| Story/Novel | Complete narrative |
| Script | INT./EXT., scene headings |

---

## Evaluation Criteria

Score 0-100:
- Structure (25%)
- Characters (20%)
- Scene Description (20%)
- Dialogue (15%)
- Pacing (10%)
- Visual Potential (10%)

Grades:
- 90-100: Excellent
- 70-89: Good
- 50-69: Fair
- 0-49: Poor

---

## Standard Script Format

```
【场景编号】 场景名 - 内/外景 - 时间 - 天气

【场景描述】
详细描述

【角色】
- 角色1：状态
- 角色2：状态

【动作】
动作描述

【对话】
角色1：台词
角色2：台词

【情感/氛围】
情感基调
```
