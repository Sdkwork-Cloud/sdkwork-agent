# Product Architecture

Video director product architecture.

---

## Overview

The video director skill provides a complete workflow from idea to film project.

---

## Core Modules

1. **Script Intelligence** - Input processing, evaluation, optimization
2. **Character Design** - Character appearance, personality, costume
3. **Location Design** - Scene settings, atmosphere, lighting
4. **Prop Design** - Key props, appearance, significance
5. **Storyboard Creation** - Scenes and shots
6. **Sound Design** - Ambience, SFX, music
7. **Output Integration** - Full project or modular output

---

## Modular Output Modes

- Full Project
- Script Only
- Characters Only
- Props Only
- Locations Only
- Storyboard Only
