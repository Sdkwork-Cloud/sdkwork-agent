# 场景设计指南

## 概述

场景设计是视频创作的重要环节，决定了故事发生的环境和氛围。本指南提供场景设计的最佳实践和建议。

---

## 场景要素

每个场景设计应包含：

| 要素 | 说明 | 示例 |
|------|------|------|
| **名称** | 场景名称 | "秋日公园长椅" |
| **类型** | 室内/室外 | 室外 |
| **场景类型** | 自然/都市/住宅/工作等 | 自然场景 |
| **时间** | 一天中的时间 | 下午 |
| **天气** | 天气状况 | 晴朗 |
| **季节** | 季节 | 秋 |
| **氛围** | 情感基调 | 温馨、宁静 |
| **灯光风格** | 灯光类型 | 自然光，金色午后阳光 |
| **色调** | 颜色风格 | 暖色调，金黄色、橙色、棕色 |
| **空间布局** | 详细描述 | 公园中央的一片开阔草地... |
| **核心特征** | 标志性元素 | 老旧的木质长椅、高大的梧桐树... |
| **视觉描述** | 简短描述 | 秋日午后的公园... |

## 场景类型

### 自然场景
- 森林、山脉、海滩、草原、湖泊、河流、瀑布
- 强调自然元素、天气变化
- 使用自然光

### 都市场景
- 街道、写字楼、咖啡馆、酒吧、商场、地铁站
- 强调建筑、人流、城市氛围
- 可用人工灯光

### 住宅场景
- 客厅、卧室、厨房、书房、浴室、阳台
- 强调生活气息、个人风格
- 温馨、私密的氛围

### 工作场景
- 办公室、实验室、工作室、工厂、医院、教室
- 强调功能性、专业氛围
- 明亮、实用的灯光

### 奇幻/科幻场景
- 魔法森林、太空站、未来城市、古代宫殿、异世界
- 强调想象力、特殊视觉效果
- 可用创意灯光

### 历史场景
- 古代街道、中世纪城堡、民国时期、80年代、古代宫殿
- 强调年代感、历史细节
- 符合时代的灯光

## 最佳实践

### ✅ 推荐做法

1. **与故事匹配**
   - 场景要服务于故事
   - 氛围要符合情感基调

2. **细节丰富**
   - 添加足够的视觉细节
   - 包括标志性元素

3. **合理的时间和天气**
   - 时间和天气要符合故事逻辑
   - 可以利用天气创造氛围

4. **考虑光照**
   - 不同时间的光线不同
   - 利用光线创造戏剧效果

### ❌ 避免的做法

1. **过于模糊**
   - 不要只说"公园"，要说"秋日午后的公园"
   - 要有具体的细节

2. **矛盾的元素**
   - 避免同时有雪和花
   - 时间和天气要一致

3. **与故事不符**
   - 场景要符合故事的类型和基调
   - 不要在喜剧片里用过于黑暗的场景

## 场景设计示例

### 示例 1：秋日公园长椅
```
名称：秋日公园长椅
类型：室外
场景类型：自然场景
时间：下午
天气：晴朗
季节：秋
氛围：温馨、宁静
灯光风格：自然光，金色午后阳光
色调：暖色调，金黄色、橙色、棕色
空间布局：公园中央的一片开阔草地，中间有一条蜿蜒的小径，小径旁有一张木质长椅，周围是高大的梧桐树，地上散落着金黄的落叶
核心特征：老旧的木质长椅、高大的梧桐树、地上散落的落叶、远处的喷泉
视觉描述：秋日午后的公园，金色阳光透过梧桐树叶洒下，木质长椅上散落着金黄的落叶
```
