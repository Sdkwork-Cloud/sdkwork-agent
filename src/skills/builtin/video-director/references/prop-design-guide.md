# 道具设计指南

## 概述

道具设计是视频创作的重要环节，可以增强故事的真实感和视觉效果。本指南提供道具设计的最佳实践和建议。

---

## 道具要素

每个道具设计应包含：

| 要素             | 说明               | 示例                                 |
| ---------------- | ------------------ | ------------------------------------ |
| **名称**         | 道具名称           | "旅行相册"                           |
| **类别**         | 道具类别           | "信物/纪念品"                        |
| **角色**         | 道具角色           | STORY, ATMOSPHERE, CHARACTER, ACTION |
| **描述**         | 简短描述           | "棕色皮质封面的旧相册"               |
| **外观**         | 外观细节           | 材质、颜色、状态、特征               |
| **故事意义**     | 在故事中的作用     | "承载着小明的旅行回忆"               |
| **交互方式**     | 角色如何使用       | "小明从背包中拿出"                   |
| **视觉标签**     | 视觉风格标签       | "怀旧"、"温馨"                       |
| **参考图提示词** | 生成参考图的提示词 | ...                                  |

## 道具类别

### 日常用品

- 咖啡杯、背包、手机、钥匙、书籍、钱包
- 贴近生活，增加真实感

### 武器/工具

- 剑、枪、工具箱、魔法杖、刀具
- 推动动作场景

### 魔法/科幻物品

- 魔法水晶、时光机、外星装置、科技设备
- 奇幻/
科幻题材

### 信物/纪念品
- 项链、照片、戒指、旅行相册、信物
- 承载情感，推动剧情

### 科技设备
- 笔记本电脑、平板、智能手表、VR设备
- 现代/未来题材

### 食物/饮料
- 咖啡、红酒、蛋糕、水果、美食
- 增加生活气息，塑造场景

### 家具/装饰
- 吊灯、挂画、地毯、钟表、装饰
- 塑造场景氛围

### 交通工具
- 汽车、自行车、飞船、马、摩托车
- 推动场景转换

## 外观细节

### 材质
- 皮质、木质、金属、塑料、布料、玻璃、石头
- 描述道具的质感

### 颜色
- 棕色、黑色、白色、红色、蓝色、金色
- 描述道具的颜色

### 状态
- 全新、良好、旧、破旧、有划痕、有污渍
- 描述道具的使用状态

### 特征
- 烫金文字、特殊标记、划痕、污渍、独特设计
- 描述道具的标志性特征

## 最佳实践

### ✅ 推荐做法

1. **与故事相关**
   - 道具要服务于故事
   - 要有意义，不要只是装饰

2. **细节丰富**
   - 添加足够的视觉细节
   - 描述材质、颜色、状态

3. **合理的状态**
   - 道具的状态要符合故事
   - 旧道具要有使用痕迹

4. **考虑交互**
   - 角色如何使用道具
   - 道具如何影响角色

### ❌ 避免的做法

1. **过于普通**
   - 道具要有特点
   - 不要只是"一个杯子"，要说"一个有裂痕的旧陶瓷杯"

2. **与故事不符**
   - 道具要符合时代和背景
   - 不要在古代出现手机

3. **没有意义**
   - 每个道具都要有存在的理由
   - 不要添加无关的道具

## 道具设计示例

### 示例 1：旅行相册
```
名称：旅行相册
类别：信物/纪念品
角色：STORY
描述：棕色皮质封面的旧相册，记录了小明的欧洲之旅
外观：
  材质：皮质
  颜色：棕色
  状态：旧，有使用痕迹
  特征：封面上有烫金的 'Travel Memories' 字样
故事意义：承载着小明的旅行回忆，推动两人情感发展
交互方式：小明从背包中拿出、翻开相册、指向照片
视觉标签：怀旧、温馨
参考图提示词：一本旧的棕色皮质旅行相册，封面有烫金文字，有些磨损，内页贴满旅行照片，温馨怀旧，特写镜头，电影质感，4K，高质量
```
