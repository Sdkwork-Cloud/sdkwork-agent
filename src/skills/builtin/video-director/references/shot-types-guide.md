# 镜头类型指南

## 概述

镜头类型（Shot Scale）是视频创作的基础元素，决定了画面中包含多少内容。本指南介绍各种镜头类型及其使用场景。

---

## 镜头类型（Shot Scales）

### 1. 极远景（EXTREME_WIDE）

**描述:** 展示非常广阔的场景，人物在画面中很小或不可见。

**适用场景:**
- 建立场景
- 展示宏大的环境
- 史诗感的画面

**提示词示例:**
```
极远景，extreme wide shot, establishing shot
```

**示例:**
```
极远景，壮丽的山脉，云海翻涌，自然风光，电影质感，4K，高质量
```

---

### 2. 远景（WIDE）

**描述:** 展示完整的场景，人物完整可见但在画面中较小。

**适用场景:**
- 展示人物与环境的关系
- 动作场景
- 舞蹈场景

**提示词示例:**
```
远景，wide shot, full shot
```

**示例:**
```
远景，两个人在公园长椅上聊天，展示整个公园环境，温馨氛围，4K，高质量
```

---

### 3. 全景（FULL）

**描述:** 展示人物从头到脚的完整身体。

**适用场景:**
- 人物出场
- 展示服装
- 全身动作

**提示词示例:**
```
全景，full shot, full body shot
```

**示例:**
```
全景，一个年轻人走进画面，穿着休闲夹克，电影质感，4K，高质量
```

---

### 4. 中景（MEDIUM）

**描述:** 展示人物从腰部以上的部分。

**适用场景:**
- 对话场景
- 人物互动
- 展示上半身动作

**提示词示例:**
```
中景，medium shot, waist shot
```

**示例:**
```
中景，两个人坐在长椅上聊天，展示上半身，温馨氛围，4K，高质量
```

---

### 5. 中近景（MEDIUM_CLOSE_UP）

**描述:** 展示人物从胸部以上的部分。

**适用场景:**
- 情感表达
- 重要对话
- 介于中景和近景之间

**提示词示例:**
```
中近景，medium close-up, MCU
```

**示例:**
```
中近景，人物说话时的表情，展示面部和上半身，4K，高质量
```

---

### 6. 近景（CLOSE_UP）

**描述:** 展示人物的面部和肩部。

**适用场景:**
- 情感特写
- 强调表情
- 重要时刻

**提示词示例:**
```
近景，close-up, CU
```

**示例:**
```
近景，人物微笑的表情，展示面部细节，温馨氛围，4K，高质量
```

---

### 7. 特写（EXTREME_CLOSE_UP）

**描述:** 展示非常小的细节，如眼睛、手、嘴巴等。

**适用场景:**
- 强调细节
- 悬念
- 重要物体

**提示词示例:**
```
特写，extreme close-up, ECU
```

**示例:**
```
特写，人物的眼睛，展示瞳孔的细节，情感表达，4K，高质量
```

---

### 8. 过肩镜头（OVER_THE_SHOULDER）

**描述:** 从一个人物的肩膀后面看向另一个人物。

**适用场景:**
- 对话场景
- 展示人物关系
- 采访场景

**提示词示例:**
```
过肩镜头，over-the-shoulder shot, OTS
```

**示例:**
```
过肩镜头，从一个人的肩膀后面看向另一个人，对话场景，4K，高质量
```

---

## 相机角度（Camera Angles）

### 1. 平视（EYE_LEVEL）

**描述:** 相机与人物眼睛在同一高度。

**适用场景:**
- 大多数场景
- 自然、真实的感觉

---

### 2. 低角度（LOW_ANGLE）

**描述:** 相机从下往上拍摄。

**适用场景:**
- 强调人物的力量
- 英雄感
- 压迫感

---

### 3. 高角度（HIGH_ANGLE）

**描述:** 相机从上往下拍摄。

**适用场景:**
- 展示人物的脆弱
- 俯瞰场景
- 全知视角

---

### 4. 鸟瞰（BIRDS_EYE）

**描述:** 从正上方垂直往下拍摄。

**适用场景:**
- 展示布局
- 地图式视角
- 抽象画面

---

### 5. 荷兰角（DUTCH_ANGLE）

**描述:** 相机倾斜拍摄。

**适用场景:**
- 不安感
- 紧张
- 超现实

---

## 镜头类型选择指南

| 叙事目的 | 推荐镜头类型 |
|---------|-------------|
| 建立场景 | EXTREME_WIDE, WIDE |
| 展示动作 | WIDE, FULL |
| 对话交流 | MEDIUM, MEDIUM_CLOSE_UP, OVER_THE_SHOULDER |
| 表达情感 | CLOSE_UP, EXTREME_CLOSE_UP |
| 强调细节 | EXTREME_CLOSE_UP |
| 展示关系 | OVER_THE_SHOULDER, TWO_SHOT |

---

## 最佳实践

### ✅ 推荐做法

1. **根据叙事目的选择镜头类型**
2. **在对话场景中交替使用中景和近景**
3. **用远景或极远景开始新场景**
4. **用特写强调重要情感或细节**

### ❌ 避免的做法

1. **不要过度使用特写**
2. **避免在对话场景中一直使用同一镜头类型**
3. **不要在快速剪辑中使用过多不同的镜头类型**
