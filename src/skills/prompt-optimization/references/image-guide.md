# Image Prompt Optimization Guide

## Quick Reference

### Quality Modifiers
Always add to improve results:
- `masterpiece` - Highest quality
- `best quality` - High quality
- `highly detailed` - More details
- `8k` / `4k` - Resolution hint
- `sharp focus` - Clear image
- `professional photography` - Photo style

### Style Keywords

**Photorealistic:**
- photorealistic, realistic, lifelike
- professional photography, DSLR
- studio lighting, natural lighting

**Artistic:**
- oil painting, watercolor, sketch
- digital art, concept art
- anime, manga, cartoon

**Atmosphere:**
- cinematic, dramatic lighting
- moody, atmospheric
- vibrant colors, muted tones

### Composition

**Framing:**
- close-up, medium shot, wide shot
- portrait, landscape
- aerial view, bird's eye view

**Camera:**
- 35mm lens, 50mm lens, wide angle
- depth of field, bokeh
- rule of thirds, centered

### Example Transformations

**Before:**
```
a cat sitting
```

**After:**
```
masterpiece, best quality, highly detailed, a fluffy orange cat sitting on a windowsill, soft natural lighting coming through the window, photorealistic, 8k, sharp focus, cozy atmosphere
```

## Tips

1. **Order matters**: Quality tags first, then subject, then details
2. **Be specific**: "fluffy orange cat" > "cat"
3. **Add lighting**: Always describe the light source
4. **Include setting**: Where is the subject?
5. **Use commas**: Separate distinct concepts with commas
